# Pranee Properties — Website Starter (GitHub Pages)

This is a clean, white, black-text starter website for **Pranee Properties**.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png` + `assets/logo@2x.png` (from your latest logo)
- `assets/hero-placeholder.png` (replace with a real hero image/video poster)

## Quick start
1. Create a new GitHub repo (e.g. `praneeproperties-site`).
2. Upload these files to the repo root.
3. In GitHub: **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / root
4. Add a custom domain in Pages (e.g. `praneeproperties.com`) — GitHub may create a `CNAME` file automatically.
